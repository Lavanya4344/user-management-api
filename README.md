# User Management API

**Backend Assignment for BuyerForeSight**

## Tech Stack
- Python 3.12
- FastAPI
- JSON file storage

## How to Run
1. Create virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate

2.Install dependencies:
pip install fastapi uvicorn

3.Run server:
uvicorn main:app --reload

4.Open Swagger docs:
http://127.0.0.1:8000/docs
